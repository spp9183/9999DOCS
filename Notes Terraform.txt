HOW TO BRING A MANUALLY CREATED CONFIG INTO TERRAFORM MANAGED STATE ==============

1) CREATE Terraform configuration manually with  with required Resources and Arguments 
2) Use Import function as below, to create statefile for those resources (sample for RG/Vnet/Subnet resources)
terraform import azurerm_resource_group.rg1 /subscriptions/$(az account show --query id -o tsv)/resourceGroups/RG1
terraform import azurerm_virtual_network.vnet1 /subscriptions/$(az account show --query id -o tsv)/resourceGroups/RG1/providers/Microsoft.Network/virtualNetworks/Vnet1
terraform import azurerm_subnet.subnet1 /subscriptions/$(az account show --query id -o tsv)/resourceGroups/RG1/providers/Microsoft.Network/virtualNetworks/Vnet1/subnets/Subnet1
terraform import azurerm_subnet.subnet2 /subscriptions/$(az account show --query id -o tsv)/resourceGroups/RG1/providers/Microsoft.Network/virtualNetworks/Vnet1/subnets/Subnet2
terraform import azurerm_subnet.routeserversubnet /subscriptions/$(az account show --query id -o tsv)/resourceGroups/RG1/providers/Microsoft.Network/virtualNetworks/Vnet1/subnets/RouteServerSubnet
3) Run terraform plan - response should be no change
